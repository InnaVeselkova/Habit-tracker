# Веб приложение трекер полезных привычек

## Описание проекта

Трекер полезных привычек позволяет пользователей создавать свои полезные привычки и управлять ими. Предусморена возможностьь устанавлмвать дату привычки, время ее выполнения, система вознаграждений в виде начисления баллов. Пользователь также может вдохновляться и искать новые привычки, просматривая профили других пользователей.

## Инструкции по запуску проекта

Предварительные требования: 

Установите необходимые зависимости:

pip install -r requirements.txt

Как запустить проект:

1.Клонируйте репозиторий и перейдите в папку проекта education

git clone https://github.com/InnaVeselkova/Habit-tracker.git

и перейдите в директорию проекта

cd habit_tracker

2.Создайте файл .env и заполните его переменными 

3.Для запуска проекта используйте команду

python manage.py runserver

Сервер запустится по адресу: http://127.0.0.1:8000/
